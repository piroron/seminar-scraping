+++

@title[インストール時注意点]

### インストール時

* `for All Users`は、チェックを**外す**
* `Add Python Path`には、チェックを**付ける**

+++

@title[動作確認とか]

### 起動確認

* `Windows +　R`を押し、`cmd`と入力

> python --version

と入力。エラーが出なければOK。

* `python --version`と入力  
エラーが出なければOK

* pipのバージョンアップ  
> python -m pip install --upgrade pip

pipは、Pythonのモジュールを管理するためのツール。
